# How to Install Street Thrift Apparel App

The Street Thrift Apparel inventory system is a **Progressive Web App (PWA)** that can be installed on your phone or desktop like a native app. Once installed, it works offline and syncs automatically when you're online.

---

## 📱 Install on Android Phone

1. **Open the app** in Chrome browser: https://thriftapparel.app (or your domain)
2. Look for the **"Add to Home Screen"** or **"Install"** prompt at the bottom of the screen
3. Tap **"Install"** or **"Add"**
4. The app icon will appear on your home screen
5. Tap the icon to open the app like any other app

**Alternative method:**
1. Open the app in Chrome
2. Tap the **3-dot menu** (⋮) in the top right
3. Select **"Add to Home screen"** or **"Install app"**
4. Tap **"Add"** or **"Install"**

---

## 📱 Install on iPhone (iOS)

1. **Open the app** in Safari browser: https://thriftapparel.app (or your domain)
2. Tap the **Share button** (square with arrow pointing up) at the bottom
3. Scroll down and tap **"Add to Home Screen"**
4. Tap **"Add"** in the top right
5. The app icon will appear on your home screen
6. Tap the icon to open the app

**Note:** On iPhone, you must use Safari browser (not Chrome) to install the app.

---

## 💻 Install on Windows Desktop

### Using Chrome:
1. **Open the app** in Chrome browser: https://thriftapparel.app (or your domain)
2. Look for the **install icon** (⊕) in the address bar (right side)
3. Click the install icon
4. Click **"Install"** in the popup
5. The app will open in its own window
6. A shortcut will be added to your desktop and Start menu

**Alternative method:**
1. Open the app in Chrome
2. Click the **3-dot menu** (⋮) in the top right
3. Select **"Install Street Thrift Apparel..."**
4. Click **"Install"**

### Using Edge:
1. **Open the app** in Edge browser: https://thriftapparel.app (or your domain)
2. Click the **3-dot menu** (...) in the top right
3. Select **"Apps"** → **"Install this site as an app"**
4. Click **"Install"**
5. The app will open in its own window

---

## 💻 Install on Mac Desktop

### Using Chrome:
1. **Open the app** in Chrome browser: https://thriftapparel.app (or your domain)
2. Look for the **install icon** (⊕) in the address bar (right side)
3. Click the install icon
4. Click **"Install"** in the popup
5. The app will open in its own window
6. A shortcut will be added to your Applications folder

### Using Safari:
1. **Open the app** in Safari browser: https://thriftapparel.app (or your domain)
2. Click **"File"** in the menu bar
3. Select **"Add to Dock"**
4. The app icon will appear in your Dock

## ✅ Benefits of Installing the App

1. **Works Offline** - Continue working even without internet
2. **Faster Loading** - App loads instantly from your device
3. **Auto Sync** - Changes sync automatically when online
4. **Native Feel** - Works like a regular app, not a website
5. **Push Notifications** - Get alerts for important updates (if enabled)
6. **No Browser Clutter** - Runs in its own window without browser tabs

---

## 🔄 How Syncing Works

- **Online:** All changes save immediately to the cloud
- **Offline:** Changes save locally on your device
- **Auto Sync:** When you go back online, all offline changes sync automatically
- **Real-time:** Multiple users see updates in real-time

## 🆘 Troubleshooting

### "Install" button doesn't appear:
- Make sure you're using a supported browser (Chrome, Edge, Safari)
- Try refreshing the page
- Check that you're on the correct website URL

### App won't install on iPhone:
- You must use Safari browser (not Chrome)
- Make sure iOS is updated to the latest version

### Offline mode not working:
- Make sure you opened the app at least once while online
- Check that you have storage space on your device

### Data not syncing:
- Check your internet connection
- Close and reopen the app
- If still not syncing, contact support
