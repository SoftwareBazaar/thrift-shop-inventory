## ğŸ‘¥ Setting Up Your Team

### **1. Add Stalls**

1. Go to **Users** page
2. Click **"Add Stall"**
3. Enter:
   - **Stall Name**: e.g., "Chuka Town"
   - **Location**: e.g., "Downtown Market"
4. (Optional) Assign a manager
5. Click **"Create Stall"**

### **2. Add Users**

1. Go to **Users** page
2. Click **"Add User"**
3. Enter:
   - **Username**: e.g., "manager1"
   - **Password**: Create a strong password
   - **Full Name**: e.g., "John Doe"
   - **Role**: "User" or "Admin"
   - **Stall**: Select which stall (optional)
4. Click **"Create User"**

### **3. Share Login with Staff**

Give each operator their own login credentials.

## ğŸ’° Recording Sales

### **For Staff:**

1. Click **"Record New Sale"**
2. Select item from dropdown
3. Enter quantity
4. Select payment type:
   - **Cash**: Immediate payment
   - **Mobile**: M-Pesa, etc.
   - **Credit**: Payment later (admin only)
5. Click **"Record Sale"**

### **For Admin:**

- All staff can record sales
- Only admin can manage credit sales
- Track all sales in **Sales** page
